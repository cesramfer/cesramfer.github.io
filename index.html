Health Application

Apple Watch

I programmed a health application for the Apple Watch, the new smartwatch developed by Apple. The application is focused on improving the health of the user by applying game mechanics and game design techniques to motivate people to achieve their goals. The user has to create a character and complete different tasks to improve his characteristics, such as running, math or memory tasks. I have used all the features in the device, for example the 3D touch to detect different pressures in the touch screen, the accelerometer and the heart rate sensor. To program the application, I used Swift 2, WatchKit and HealthKit.

Slam Junk

Unreal Engine 4

Slam Junk was my second game with Unreal Engine 4. This is an arena game of robots fighting to collect batteries. In this game I worked in the mechanics and programming the animations. This project was made with a group of programmers, artist, designers and animators.

World Wide Wreck

Unreal Engine 4

World Wide Wreck was my first game with Unreal Engine 4. This is an old school beat em up Vikings game with 2 levels. In this game I worked in the mechanics, programming the animation and the user interface. This game was made with a group of programmers and artists.

CesEngine

OpenGL

CesEngine is a personal engine programmed with C++ and OpenGL to learn how work an engine. In this project I implemented some features such as shadow mapping or blur.
